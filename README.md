# Lost & Found Website

## Features
- User Registration/Login
- Add Lost Items
- Add Found Items
- Upload Images
- Search Items
- Dashboard
- Bootstrap UI

## Run Locally

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Open:
http://127.0.0.1:8000

Admin:
http://127.0.0.1:8000/admin
